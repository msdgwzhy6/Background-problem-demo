# 后台常见问题 demo 展示

> https://bosens-china.github.io/Background-problem-demo/dist/

演示了三种情况

1. 虚拟列表，解决数据返回数量不固定，浏览器卡顿

2. 全局 loading 管理

3. 自定义组由的显示缓存（例如列表进入详情，返回列表状态信息保存）

## 目录划分

```sh
├── src
    ├── view
        ├── fictitious                虚拟列表
        ├── keep-alive                自定义组由的显示缓存
        ├── loading                   全局loading管理
```
